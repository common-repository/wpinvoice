Ce plugin permet de g&eacute;rer des factures dans wordpress.

== Description ==

Vous pouvez g&eacute;rer vos factures depuis l'administration de wordpress. Vous cr&eacute;ez des magasins, vous affectez des gestionnaires &agrave; ces magasins. Puis vous cr&eacute;ez des factures que vous affectez &agrave; un utilisateur. Vous pourrez voir vos factures en pdf ou les afficher directement dans votre navigateur. L'utilisateur de son c&ocirc;t&eacute; pourra se connecter &agrave; son compte pour visualiser ses factures

== Installation ==

L'installation du plugin peut se faire de 2 fa&ccedil;ons :

* M&eacute;thode 1

1. T&eacute;l&eacute;chargez le plugin "wpinvoice" depuis wordpress
2. Uploader le dossier du plugin que vous aurez au pr&eacute;alable d&eacute;zipper dans le r&eacute;pertoire `/wp-content/plugins/`
3. Activer le plugin dans le menu `Extensions` de Wordpress

* M&eacute;thode 2

1. Rechercher le plugin "wpinvoice" &agrave; partir du menu "Extension" de Wordpress
2. Lancer l'installation du plugin
